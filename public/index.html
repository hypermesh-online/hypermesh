Start Building Today

Complete infrastructure ready for production deployment

1

Clone Repositories

Get the complete HyperMesh ecosystem from GitHub

git clone https://github.com/hypermesh-online/trustchain.git
git clone https://github.com/hypermesh-online/hypermesh.git
git clone https://github.com/hypermesh-online/stoq.git
2

Build Infrastructure

Compile and configure the core services

cd trustchain && cargo build --release
cd ../hypermesh && cargo build --release
cd ../stoq && cargo build --release
3

Deploy Services

Start the distributed infrastructure

# Terminal 1: TrustChain CA
cargo run --bin trustchain-server

# Terminal 2: HyperMesh Platform  
cargo run --bin hypermesh-server

# Terminal 3: STOQ Transport
cargo run --bin stoq-server
