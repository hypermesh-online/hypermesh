Quick Start Guide

Get Hypermesh Nexus running in under 5 minutes!

üöÄ Installation

Prerequisites

# Ensure you have Rust 1.75+ rustup update stable # Install system dependencies (Ubuntu/Debian) sudo apt-get update sudo apt-get install -y build-essential pkg-config clang llvm # For eBPF support (optional but recommended) sudo apt-get install -y linux-headers-$(uname -r) libelf-dev

Get the Code

git clone https://github.com/your-org/hypermesh.git cd hypermesh

./deploy.sh deploy --type systemd --size 7 --env prod

Monitor Performance

üîç Troubleshooting

Build Errors

# Clean and rebuild cd core cargo clean cargo build --release

Port Conflicts

# Check for port usage netstat -tulpn | grep :8080 netstat -tulpn | grep :8081 # Deploy with different ports ./deploy.sh deploy --type local --size 3 --config custom-ports.toml

Permission Issues

# For eBPF features (requires root) sudo cargo run --bin nexus-test -- unit --component ebpf # For systemd deployment sudo ./deploy.sh deploy --type systemd --size 3
