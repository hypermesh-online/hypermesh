üîß What Gets Deployed

Node Configuration

  • Cluster: hello-world-cluster
  • Nodes: 3 (local processes)
  • Environment: Development
  • Security: Basic TLS certificates
  • Monitoring: Built-in metrics enabled

Resource Requirements

Per Node:
  • CPU: 1 core
  • RAM: 1GB
  • Storage: 10GB
  • Network: Local loopback
Total System:
  • CPU: 3 cores
  • RAM: 3GB
  • Storage: 30GB

üìä Available Scripts

./deploy.sh

Main deployment script

# Default deployment ./deploy.sh # With verbose output ./deploy.sh --verbose # Interactive step-by-step ./deploy.sh --step-by-step # Custom size ./deploy.sh --size 5

./status.sh

Cluster status and monitoring

# Quick status check ./status.sh # Detailed status with metrics ./status.sh --detailed # Watch real-time updates ./status.sh --watch

./demo.sh

Interactive demonstrations

# Run full demo with explanations ./demo.sh # Quick silent demo ./demo.sh --quiet # Demo specific component ./demo.sh --component transport

./cleanup.sh

Stop cluster and cleanup

# Stop cluster and clean up ./cleanup.sh # Keep data for next run ./cleanup.sh --keep-data # Force cleanup if processes stuck ./cleanup.sh --force
